# Utility: Confidence Interval Helper

Computes normal or t-based confidence interval.

## Usage

``` r
tc_confint(mean, se, level = 0.95, df = NULL)
```

## Arguments

- mean:

  Numeric mean

- se:

  Standard error

- level:

  Confidence level

- df:

  Degrees of freedom (optional)

## Value

Numeric vector: lower, upper
