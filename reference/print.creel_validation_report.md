# Print a creel_validation_report

Renders a colour-coded cli summary of the aggregated validation report.

## Usage

``` r
# S3 method for class 'creel_validation_report'
print(x, ...)
```

## Arguments

- x:

  A `creel_validation_report` object returned by
  [`validation_report()`](https://chrischizinski.github.io/tidycreel/reference/validation_report.md).

- ...:

  Ignored.

## Value

`x`, invisibly.
