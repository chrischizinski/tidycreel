# Print a creel_data_validation result

Renders a colour-coded cli summary grouped by table and column. Counts
pass/warn/fail verdicts in a header line.

## Usage

``` r
# S3 method for class 'creel_data_validation'
print(x, ...)
```

## Arguments

- x:

  A `creel_data_validation` object returned by
  [`validate_creel_data()`](https://chrischizinski.github.io/tidycreel/reference/validate_creel_data.md).

- ...:

  Ignored.

## Value

`x`, invisibly.
