# Repo Map — tidycreel

## Purpose
R package for creel survey design, estimation, visualisation, and reporting.
Tidy interface over the `survey` package for fisheries biologists.

## Main directories
- `R/` — all package functions
- `tests/testthat/` — test suite
- `vignettes/` — user-facing articles (rendered to pkgdown site)
- `inst/rmarkdown/templates/` — creel-dashboard flexdashboard template
- `data/` — exported example datasets
- `data-raw/` — scripts that generate example datasets
- `man/` — roxygen2-generated Rd files (do not hand-edit)
- `docs/` — pkgdown site output (gitignored or gh-pages branch)
- `.planning/` — gitignored planning docs

## Key files
- `DESCRIPTION` — package metadata, dependencies, version
- `NAMESPACE` — auto-generated by roxygen2; do not hand-edit
- `_pkgdown.yml` — pkgdown site structure and navigation
- `NEWS.md` — user-facing changelog
- `README.Rmd` / `README.md` — GitHub landing page
- `.Rbuildignore` — files excluded from R CMD check
- `justfile` — command surface

## Build/test commands
See `justfile`. Verification contract:
- `just check` — 0 errors, 0 warnings (rcmdcheck --as-cran)
- `just site` — pkgdown builds cleanly

## Known constraints
- Do not hand-edit `man/` or `NAMESPACE`
- `.Rbuildignore` must cover any new non-package root files
- CRAN submission pending — one irreducible NOTE expected (new submission)
- Do not push or publish to CRAN without explicit instruction
