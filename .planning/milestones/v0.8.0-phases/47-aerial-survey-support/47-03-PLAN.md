example_aerial_counts dataset structure:
```r
# One instantaneous count per sampling day — each row is one aerial overflight
# Columns: date (Date), day_type (character), n_anglers (integer)
# Size: 14-20 rows (sampling days across a summer season)
# Realistic values: summer Nebraska walleye fishery
#   Weekdays: 15-40 anglers; Weekends: 40-80 anglers
# Some variability between days to produce non-trivial Var(N_hat)
example_aerial_counts <- data.frame(
  date     = as.Date(c("2024-06-03", "2024-06-04", ...)),
  day_type = c("weekday", "weekday", "weekend", ...),
  n_anglers = c(22L, 18L, 45L, 38L, ...),
  stringsAsFactors = FALSE
)
stopifnot(all(example_aerial_counts$n_anglers > 0))
usethis::use_data(example_aerial_counts, overwrite = TRUE)
```
