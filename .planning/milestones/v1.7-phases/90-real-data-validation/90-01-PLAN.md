<objective>
Create six static CSV fixture files in `inst/extdata/calamus-2016/` that represent the Calamus 2016 bus-route survey data, plus a pre-computed reference-outputs.csv derived by running the tidycreel pipeline on those same fixtures.

Purpose: The validation script in Plan 02 reads these files and checks that pipeline output matches reference-outputs.csv within 0.1% relative tolerance. Because the reference values are computed FROM the fixtures (not from real archived data), the validation proves pipeline determinism and correctness, not calibration against external ground truth.

Output: Six CSV files in inst/extdata/calamus-2016/ representing a realistic but synthetic Calamus 2016 bus-route survey with all known edge cases: duplicated interview UIDs, three-value catch_type, species code 86 vs 862.
</objective>

```r
make_row <- function(label, result) {
  e <- result$estimates
  data.frame(
    estimand = label,
    estimate = e$estimate,
    se       = e$se,
    ci_lower = e$ci_lower,
    ci_upper = e$ci_upper,
    stringsAsFactors = FALSE
  )
}
ref <- rbind(
  make_row("effort_total",  eff),
  make_row("catch_total",   cat_est),
  make_row("harvest_total", harv)
)

cat("Reference outputs written.\n")
print(ref)
```
