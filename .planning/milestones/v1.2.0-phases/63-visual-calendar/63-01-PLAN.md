    **1. Private helper: `make_day_abbrevs(day_type_labels)`**
    Implements the collision-resolution loop from RESEARCH.md Pattern 4:
    - `unique_labels <- unique(day_type_labels)`
    - Loop `k` from 2 to `max(nchar(unique_labels))`
    - Each iteration: `abbrevs <- toupper(substr(unique_labels, 1L, k))`

    - Return `stats::setNames(abbrevs, unique_labels)` (named vector: label -> abbreviation)

    **2. Private helper: `build_month_grid(month_dates, cell_lookup, mode)`**
    - `month_dates`: Date vector of all calendar days in the month (from floor_date to ceiling_date - 1)
    - `cell_lookup`: named list/vector mapping `as.character(date)` -> cell content string (NULL/NA for non-sampled)
    - `mode`: `"ascii"` (returns character vector of text lines) or `"pandoc"` (returns character vector of markdown lines)

    Algorithm:
    - `month_start <- lubridate::floor_date(month_dates[1], "month")`
    - `pad_left <- lubridate::wday(month_start, week_start = 7L) - 1L` (Sunday=1, so pad_left=0 if starts Sunday)
    - `all_dates <- seq(month_start, lubridate::ceiling_date(month_start, "month") - lubridate::days(1), by = "day")`
    - Pad to full weeks: prepend `pad_left` NAs, append NAs to reach multiple of 7
    - Chunk into rows of 7 dates

    For each date slot in the grid:
    - If NA (padding): empty cell
    - If date in `cell_lookup` and not NA: use cell content
    - Otherwise: date number only (e.g., `"15"`)

    Cell width: dynamically size to `max(nchar(cell content across all cells), 8L)` for consistent columns.

    ASCII mode output lines:
    - Header: `"| Sun      | Mon      | Tue      | Wed      | Thu      | Fri      | Sat      |"` (padded to cell width)
    - Separator: `"|----------|----------|..."`
    - Each week row: pipe-separated cells, padded. For bus-route (two-line cells), the date number appears on the first sub-row, abbreviation on second, circuit on third — rendered as 3 text lines per week row with leading pipe per sub-row.
    - Mark `@noRd`

    Pandoc mode output lines:
    - Header: `"| Sun | Mon | Tue | Wed | Thu | Fri | Sat |"`
    - Separator: `"|-----|-----|...|"`
