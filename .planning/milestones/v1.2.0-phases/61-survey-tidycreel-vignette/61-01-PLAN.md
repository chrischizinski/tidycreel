Core tidycreel workflow for an instantaneous count design with interviews:
```r
library(tidycreel)
data(example_calendar)    # date, day_type columns
data(example_counts)      # date, count columns
data(example_interviews)  # date, catch_total, hours_fished, catch_kept,
                          # trip_status, trip_duration columns

design <- creel_design(example_calendar, date = date, strata = day_type)
design <- add_counts(design, example_counts)
design <- add_interviews(design, example_interviews,
  catch    = catch_total,
  effort   = hours_fished,
  harvest  = catch_kept,
  trip_status   = trip_status,
  trip_duration = trip_duration
)
effort_est     <- estimate_effort(design)
cpue_est       <- estimate_catch_rate(design)
total_catch    <- estimate_total_catch(design)
```

Equivalent raw survey package calls (what tidycreel wraps internally):
- `creel_design()` + `add_counts()` -> builds a `survey::svydesign()` with stratified sampling weights derived from the calendar
- `estimate_effort()` -> `survey::svymean()` on count column x h_open, or `survey::svytotal()`
- `estimate_catch_rate()` -> `survey::svyratio(numerator = ~catch, denominator = ~effort)` on interview data (ratio-of-means)
- `estimate_total_catch()` -> effort_estimate x cpue_estimate with delta-method variance: Var(E x C) = E^2 Var(C) + C^2 Var(E)

   ```r
   library(survey)
   # Build stratified frame: fpc = total days in each stratum
   counts_with_strata <- merge(example_counts, example_calendar, by = "date")
   N_h <- table(example_calendar$day_type)
   counts_with_strata$fpc <- as.integer(N_h[counts_with_strata$day_type])
   svy_design <- svydesign(
     ids = ~1,
     strata = ~day_type,
     fpc = ~fpc,
     data = counts_with_strata
   )
   ```
