<objective>
Implement estimate_harvest() with ratio-of-means HPUE estimation, rename validate_cpue_sample_size to shared validate_ratio_sample_size, and verify via TDD with reference tests.

Purpose: Harvest estimation (HPUE = harvest per unit effort) enables biologists to estimate the rate of kept fish, distinguished from total catch rate (CPUE). This mirrors the estimate_cpue() architecture exactly, substituting design$harvest_col for design$catch_col as the svyratio numerator.

Output: Working estimate_harvest() function with ungrouped/grouped estimation, shared sample size validation, and reference tests proving correctness against manual svyratio calculations.
</objective>
