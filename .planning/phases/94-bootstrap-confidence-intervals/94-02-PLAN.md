Chapman point estimate formula:   N_hat = ((M+1)*(n+1)) / (m+1) - 1
Petersen point estimate formula:  N_hat = (M * n) / m
Schnabel point estimate formula:  N_hat = sum(M * n) / sum(m)

Bootstrap resampling (parametric, uses stats::rbinom only — no survey package):

  Chapman/Petersen (single occasion — M, n, m are scalars):
    m_b <- stats::rbinom(B, size = n, prob = m / n)   # B replicates
    # Guard against m_b == 0 (undefined N_hat):
    m_b[m_b == 0L] <- 1L
    N_hat_b_chapman <- ((M+1)*(n+1)) / (m_b+1) - 1
    N_hat_b_petersen <- (M * n) / m_b

  Schnabel (multi-occasion — M, n, m are vectors of length k):
    # Resample each occasion independently:
    m_b_matrix <- vapply(
      seq_along(m),
      function(i) stats::rbinom(B, size = n[i], prob = m[i] / n[i]),
      numeric(B)
    )  # matrix B x k
    # Guard: replace 0s per column
    sum_m_b <- rowSums(m_b_matrix)
    sum_m_b[sum_m_b == 0L] <- 1L
    N_hat_b <- sum(M * n) / sum_m_b  # vector of length B

Percentile CI:
  alpha <- 1 - conf_level
  ci_lo_boot <- stats::quantile(N_hat_b, alpha / 2,     names = FALSE)
  ci_hi_boot <- stats::quantile(N_hat_b, 1 - alpha / 2, names = FALSE)

Adding to estimates_df BEFORE new_creel_estimates():
  estimates_df$ci_lo_boot <- ci_lo_boot
  estimates_df$ci_hi_boot <- ci_hi_boot

Storing boot_samples AFTER new_creel_estimates() returns:
