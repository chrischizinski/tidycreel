## Decisions Made

- `creel_n_camera()` implemented as a standalone function (not a `power_creel()` mode), resolving OQ-5 from STATE.md
- FPC omitted intentionally for pre-season planning, consistent with `creel_n_effort()` precedent
- Feltz-Middaugh (2025) minimums applied as advisory warnings (not hard stops) — computed n is returned unchanged even when below threshold
- Stratum classification uses case-insensitive `grepl()` partial matching: "weekday" and "weekend" substrings, not exact match
- `vapply()` used for per-stratum minimum lookup (not `sapply`), consistent with package convention
