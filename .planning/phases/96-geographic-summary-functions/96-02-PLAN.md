devtools::document() and rcmdcheck to complete the Phase 96 gate.

read the ii_ZipCode field from design$interviews, handle NA values as explicit "Unknown"
rows, and compute N + percent over all interviews (denominator includes NA).
summarize_by_county() uses zipcodeR for zip-to-county lookup with a hard cli_abort()
when the package is not installed.
