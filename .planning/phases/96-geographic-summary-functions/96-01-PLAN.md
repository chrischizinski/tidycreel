Implement summarize_boat_composition(design, schema) as the first of three geographic
summary functions for NGPC standard creel reports.

Purpose: Biologists need to produce the "Angler Boats by Month and Day Type" table for
NGPC reports. This function computes c_AnglerBoats / (c_AnglerBoats + c_NonAngBoats)
per count event, aggregated by month x day_type, from design$counts.

tests/testthat/test-creel-summaries-geographic.R.
