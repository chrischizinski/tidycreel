<objective>
Implement add_counts() function and internal survey bridge construction for attaching
instantaneous count data to creel_design objects and eagerly constructing svydesign objects.

Purpose: This is the core of the survey bridge layer -- translating domain vocabulary (creel counts)
into statistical machinery (survey::svydesign). Eager construction catches design errors at
add_counts() time when users have context about what data they are adding.

Output: Working add_counts() that attaches validated count data to creel_design and constructs
internal svydesign object. Tests proving validation catches common errors with helpful messages.
</objective>
