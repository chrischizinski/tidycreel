<objective>
Implement estimate_effort() function that wraps survey::svytotal() to produce total effort estimates with standard errors, confidence intervals, and sample sizes, returning a creel_estimates S3 object. Includes Tier 2 validation warnings for data quality issues and reference tests proving estimates match manual survey package calculations.

Purpose: This is the core estimation capability of tidycreel v2. It proves the three-layer architecture works end-to-end: user calls estimate_effort() (domain API) which internally uses the survey bridge (design$survey) to call survey::svytotal() (statistical engine), then wraps results in a creel_estimates object (domain output).

Output: Working estimate_effort() function with Tier 2 validation and reference tests confirming correctness.
