    new_creel_validation(results, tier, context):
    - results: data.frame with columns: check (character), status (character: "pass" or "fail" or "warn"), message (character)
    - tier: integer (1, 2, or 3)
    - context: character(1) describing validation context (e.g., "creel_design creation", "effort estimation")
    - Computes passed = all(results$status == "pass") automatically
    - Returns list with class "creel_validation"

    Structure:
    ```
    list(
      results = <data.frame with check, status, message>,
      tier    = <integer>,
      context = <character(1)>,

    )
    class: "creel_validation"
    ```
