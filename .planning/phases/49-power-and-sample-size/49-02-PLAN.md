    Formula (two-sample normal approximation, equal group sizes):
      ncp    = abs(delta_pct) * sqrt(n / 2) / cv_historical
      z_crit = if two.sided: qnorm(1 - alpha/2)  else: qnorm(1 - alpha)
      power  = if two.sided: pnorm(ncp - z_crit) + pnorm(-ncp - z_crit)
               else:         pnorm(ncp - z_crit)

      delta_pct    = fractional change to detect, e.g. 0.20 for 20% (numeric > 0)

    Signature: cv_from_n(type = c("effort", "cpue"), n, ...)

    Effort inverse (type = "effort"):
      Additional args: N_h, ybar_h, s2_h
      E_total = sum(N_h * ybar_h)
      CV      = sqrt(sum(N_h * s2_h) / n) / E_total
      return CV (numeric scalar)

    CPUE inverse (type = "cpue"):
      Additional args: cv_catch, cv_effort, rho = 0
      CV = sqrt((cv_catch^2 + cv_effort^2 - 2 * rho * cv_catch * cv_effort) / n)
      return CV (numeric scalar)
