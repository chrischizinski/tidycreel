    Formula (Cochran 1977 eq. 5.25, FPC omitted — standard for pre-season planning):
      E_total     = sum(N_h * ybar_h)
      V_0         = (cv_target * E_total)^2
      numerator   = sum(N_h * sqrt(s2_h))^2
      denominator = V_0 + sum(N_h * s2_h)
      n_total     = ceiling(numerator / denominator)
      w_h         = N_h / sum(N_h)          # proportional weights
      n_h         = ceiling(n_total * w_h)  # per-stratum days
      return c(n_h, total = n_total)

    Inputs (biologist-facing, not raw variances — per research recommendation):
      cv_catch   = pilot CV of catch per interview (numerator of CPUE ratio)
      cv_effort  = pilot CV of effort per interview (denominator)
      rho        = pilot correlation between catch and effort (default 0 — conservative)
      cv_target  = desired CV of the CPUE ratio estimate

    Formula (Cochran 1977, ratio estimator variance approximation):
      n = ceiling((cv_catch^2 + cv_effort^2 - 2 * rho * cv_catch * cv_effort) / cv_target^2)
      n = max(n, 1L)   # floor at 1
