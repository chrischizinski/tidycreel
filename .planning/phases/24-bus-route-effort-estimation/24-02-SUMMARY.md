## Decisions Made

- `trip_duration` omitted from bus-route test helpers: `add_interviews()` accepts it as optional; including `trip_duration = 0.0` for the zero-effort site row (n_counted=0, n_interviewed=0) would fail Tier 2 validation (< 1 minute threshold). Bus-route estimation does not use trip_duration.
- `expect_no_message(suppressWarnings())` for verbose=FALSE dispatch test: `survey::svydesign()` emits "no weights or probabilities supplied" warning that is expected/benign for bus-route data; the test should only verify no *dispatch message* is emitted by tidycreel, not full silence.

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Removed trip_duration from bus-route test helpers**
- **Found during:** Task 2 (bus-route test suite)
- **Issue:** Plan's `make_br_effort_interviews()` included `trip_duration` column in data frame and `trip_duration = trip_duration` in `add_interviews()` calls. For the zero-effort test case (site B: n_counted=0, n_interviewed=0), `trip_duration = 0.0` fails Tier 2 validation ("values less than 1 minute are unrealistic"). Bus-route effort estimation does not use trip duration.
- **Fix:** Removed `trip_duration` column from all bus-route test data frames and removed `trip_duration = trip_duration` from all `add_interviews()` calls in the bus-route section.
- **Files modified:** tests/testthat/test-estimate-effort.R
- **Verification:** `devtools::test(filter = "estimate-effort")` FAIL = 0
- **Committed in:** 81b67b5 (Task 2 commit)

**2. [Rule 1 - Bug] Changed verbose=FALSE test from expect_silent() to expect_no_message(suppressWarnings())**
- **Found during:** Task 2 (bus-route test suite)
- **Issue:** `expect_silent(estimate_effort(d))` failed because `survey::svydesign()` emits "No weights or probabilities supplied, assuming equal probability" warning during bus-route estimation. This is a benign expected warning from the survey package, not related to tidycreel dispatch behavior.
- **Fix:** Changed to `expect_no_message(suppressWarnings(estimate_effort(d)))` to only check for messages (the dispatch output), not warnings.
- **Files modified:** tests/testthat/test-estimate-effort.R
